# GAQP Atomic Nugget Admission Tests

**Status:** Canonized | **Version:** 1.0 | **Authority:** Execalc GAQP Standards Body

## The Seven Admission Tests

| # | Test | Requirement |
|---|---|---|
| 1 | Stand-Alone Test | Retains full meaning outside its source paragraph |
| 2 | Disputability Test | Can be agreed with, challenged, or qualified by a serious actor |
| 3 | Governance Test | Can be tagged, sourced, scoped, and audited |
| 4 | Activation Test | The system can determine when and where this claim should matter |
| 5 | Durability Test | Likely to remain useful beyond the moment of utterance |
| 6 | Composability Test | Can combine with other nuggets into larger reasoning structures |
| 7 | Non-Triviality Test | Carries strategic or interpretive weight |

## Application Rules

- Tests are applied in order. Failure at Test 1 terminates evaluation.
- Tests 2 and 7 are most commonly failed.
- Test 4 is the most forward-looking — asks whether the system will know when to use this claim.
- A statement passing all 7 tests enters the GAQP classification pipeline.

## Runtime Enforcement

The Nugget Runtime Enforcement Reflex (NRE-01) governs admission test execution.
It fires whenever a nugget candidate is submitted for governed memory storage.
