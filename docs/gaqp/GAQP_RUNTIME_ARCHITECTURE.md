# GAQP Runtime Architecture
**Status:** Canonized | **Version:** 1.0 | **Authority:** Execalc GAQP Standards Body

## The Six Layers

### Layer 0 — Raw Input Ingress
All formats enter through a single front door. Six input categories:
1. Human-generated language (highest signal)
2. Internal communications (sleeper category)
3. Structured reports and data
4. External signal feeds
5. Stored corpus and artifacts (compounding engine)
6. Uploaded documents (tenant onramp)

### Layer 1 — GAQP Classification Engine
The Five Diagnostic Questions run on every input signal:
1. What is asserted?
2. What is compared?
3. What is the frequency?
4. What is unknown?
5. What is at stake?

### Layer 2a — Thought Leader Bench
High-level frameworks (OODA, Blue Ocean, Porter, Christensen) contextualize
atom clusters against known strategic doctrine.

### Layer 2b — GAQP Granular Layer
Domain-specific sub-atomic context. Industry translation layers live here.

### Layer 3 — Persistent Signal Store
Critical doctrine: the corpus is not a retrieval store. It is a re-classification
engine. Prior session atoms re-enter the pipeline as first-class input signals.
The system does not load history. It re-classifies it.

### Layer 4 — Prime Directive Governance Filter
1. Value creation pathway exists
2. Supply/demand imbalance is real and addressable
3. Risk/reward is asymmetric in the executive's favor
Outputs that fail are flagged, not suppressed.

### Layer 5 — Polymorphia + Heuristics → Executive Output
Synthesized verdict — governed, cited, traceable, audit-ready.

## The Seven Output Types

| # | Type | When | Description |
|---|---|---|---|
| 1 | Synthesized Verdict | Always | Primary governed judgment. Must pass Prime Directive. |
| 2 | GAQP Atom Store Update | Always | Silent. Atoms written to corpus. |
| 3 | Precedent Citation | Triggered | Corpus match above 0.75 confidence. |
| 4 | Proactive Alert | Triggered | Passive ingestion crosses structural threshold. |
| 5 | Cross-Source Synthesis | Triggered | Multiple independent sources converge. |
| 6 | Governance Flag | Triggered | Prime Directive fails. Output delivered WITH flag. |
| 7 | Audit-Ready Corpus Log | Passive | Full audit trail. |

## Relationship to Strategic Mesh
GAQP (Layers 0-3) is the upstream classification layer that feeds the Strategic Mesh.
GAQP Classification -> Strategic Mesh (Core 7) -> Prime Directive -> Output
