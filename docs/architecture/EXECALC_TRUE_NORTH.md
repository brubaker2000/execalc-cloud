# Execalc True North

Execalc is a governed executive intelligence system that converts unstructured input into structured, auditable strategic decisions.

## Non-Negotiable System Definition
- The large language model (LLM) is the language engine. Execalc is the executive cognition layer.
- Governance sits above generation. Judgment sits above automation.
- The repository is the source of truth for both code and doctrine.

## Prime Directive Orientation
Every decision path must be evaluated through:
- Assets vs. liabilities
- Supply vs. demand
- Risk vs. reward

## Determinism Over Prompt Improvisation
- Core reasoning order is enforced in software modules, not by prompts.
- Prompts may shape language output; they may not define the system’s reasoning architecture.

## Executive Outputs
Execalc outputs must be structured enough to be:
- reviewed by leadership
- audited later
- compared across time
- stored as organizational memory

## What Execalc Is Not
- Not a prompt engineering product
- Not a generic agent playground
- Not an ungoverned chatbot layer

## Canonical Language
- Industrial-grade organizational cognition
- Governance above the LLM substrate
- Deterministic convergence toward True North
